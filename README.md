## CLUC

ClusterCreator


#### List instances
```
$ cluc vms
```

#### List templates
```
$ cluc templates
```

#### Terminate instance
```
$ cluc terminate --id <vm-id>
```
or terminate by its name
```
$ cluc terminate --name <vm-name>
```


### Development
```
$ mkvirtualenv -p python3 cluc
$ pip install -rx requirements.txt
```
